Datasets from the following websites are supported:

  • Hugging Face
  • Kaggle
  • The luxembourgish open data platform
  • OECD Data Explorer
  • World Bank Open Data
  • Google Dataset Search

Navigate to one of their datasets, copy the URL at the top of your browser, and paste it in the field above. In case of an OECD dataset, use the Data Query URL found in the Developer API category.
